WhisperJAV v1.1.2
=================

Thank you for installing WhisperJAV!

Quick Start:
-----------
1. Double-click the WhisperJAV desktop icon
2. Select your video/audio files
3. Choose processing mode
4. Click "Start Processing"

First Run:
----------
On first run, WhisperJAV will download the AI models (~3GB).
This is a one-time download.

Output Location:
---------------
By default, subtitles are saved to: ~/WhisperJAV_Output

Support:
--------
GitHub: https://github.com/meizhong986/WhisperJAV
Documentation: See GitHub README

System Requirements:
-------------------
- Windows 10/11
- NVIDIA GPU (recommended)
- 8GB+ RAM
- ~5GB disk space (including models)
